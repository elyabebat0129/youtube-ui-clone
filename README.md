# YouTube UI Clone

A responsive YouTube interface clone built with HTML, CSS and JavaScript.

This project was developed for learning purposes, focusing on responsive design, layout structuring and basic DOM interactions by recreating a YouTube-like interface.

---

## Features

* YouTube-inspired layout
* Responsive design (desktop and mobile)
* Sidebar navigation (expand/collapse)
* Mobile menu with modal interaction
* Channel page layout (banner, avatar, tabs)
* Video grid display
* Subscribe button with state persistence using localStorage

---

## Tech Stack

* HTML
* CSS (modular structure)
* JavaScript (vanilla)

---

## What I Practiced

* Responsive design (mobile-first concepts)
* Layout replication from a real-world application
* Flexbox and CSS Grid
* Separation of styles into multiple CSS files
* DOM manipulation with JavaScript
* Event handling (click interactions, toggles)
* Using localStorage to persist UI state
